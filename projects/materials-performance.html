visual systems

This visual systems project was slowly built from a prompt to an event poster using collage. It grew to incorporate several other posters utilizing variables of elements, a type treatment, 1" button, and gif. With a focus on texture and lighting quality, materials performance emphasizes the interaction between color, texture, and text.
